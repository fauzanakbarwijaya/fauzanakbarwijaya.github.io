SIPKL

Student interns' data management in a company.

Client : Pustikom Universitas Negeri Jakarta
Year : 2023

This project is a redesign and redevelopment of the old SIPKL website. SIPKL is a web-based application that functions as data management for students who are doing practical work at PUSTIKOM Universitas Negeri Jakarta. SIPKL consists of 2 parts. Namely the internship registration section for students and then the implementation of student activities such as realtime attendance and journal filling.

Process

Because this is a team project, I served as a fullstack developer in the user system section. such as :

1. Making internship student registration

I am in charge of creating an internship student registration page at PUSTIKOM Universitas Negeri Jakarta before students can access the SIPKL page.

2. Create an attendance system

Here I am tasked with creating an attendance system such as realtime attendance, permits, illnesses, absences without information, holidays and so on.

3. Create a journal filling system

I am also in charge of creating a journal filling system such as adding journal data and editing journals.
